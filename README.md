# AI Feedback Analyzer

An AI-powered system that automatically analyzes event feedback using Google technologies and Gemini AI.

---

## Problem Statement
Event feedback is collected in large volumes, but manual analysis is time-consuming and inefficient. Important insights are often ignored.

---

## Solution
This project automates feedback analysis by:
- Collecting feedback via Google Forms
- Storing responses in Google Sheets
- Aggregating feedback using Google Apps Script
- Generating AI-based summaries using Gemini AI

---

## Features
- Automated feedback collection
- Combined raw feedback view
- AI-generated summary (sentiment, positives, issues)
- Cloud-based and scalable

---

## Technologies Used
- Google Forms
- Google Sheets
- Google Apps Script
- Gemini AI API

---

## Architecture
User → Google Form → Google Sheet → Apps Script → Gemini AI → Summary Output

---

## MVP
The MVP demonstrates real-time feedback analysis with AI-generated insights displayed in Google Sheets.
